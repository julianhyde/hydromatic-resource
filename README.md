# hydromatic-resource

Compiler-checked wrappers around resource files.

# Download and build

You need Java (1.6 or higher; 1.8 preferred), git and maven (3.0.4 or later).

```bash
$ git clone git://github.com/julianhyde/hydromatic-resource.git
$ cd hydromatic-resource
$ mvn package
```
